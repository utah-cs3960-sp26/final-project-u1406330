# Archived Project Proposal And Week 13 Notes

This document preserves the older proposal and Week 13 progress narrative that previously lived in the root README. It is kept for project history only. For current behavior, read [../current_state.md](../current_state.md).

## Original Proposal Summary

The project was proposed as a simple Roblox idle game called Gem Factory, inspired by the design patterns in Grow A Garden. The core idea was to build a very simple loop with strong retention hooks:

- Idle progression where progress continues while the player is offline.
- Variable rewards through rare drops and random shop rotations.
- Timers and restocks that encourage return visits.
- Big-number progression.
- Scarcity events and limited items.
- Social comparison through visible player bases.
- A simple loop that is easy to understand and repeat.

The project was also intended as an agent-assisted development experiment. Backend logic would be covered by deterministic tests where possible, while Roblox Studio would be used to validate interactive world behavior and presentation.

## Original Theme Direction

- Players collect and break geodes instead of hatching zoo-themed rewards.
- Rewards are gems, rare minerals, crystals, and valuable resources.
- Progress should feel like building a richer gem-processing operation in a magical cavern.
- The long-term fantasy is moving from a tiny starter setup to a high-value gem empire filled with rare finds and production upgrades.

## Week 13 Snapshot

At the Week 13 checkpoint, the game world was already far enough along to support the basic loop: buy a geode, put it on a plot, wait for it to finish, crack it open, and use the result to begin passive income.

The next planned work at that point was:

- More Roblox Studio playtesting.
- Clearer placement feedback.
- Better geode cracking presentation.
- More polish on the early idle loop.
- More careful documentation of project state.
- Continued use of tests or Studio checks instead of relying only on interactive observation.
